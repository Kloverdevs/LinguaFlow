# Security Policy

## Supported Versions

Only the latest `main` branch version of LinguaFlow is officially supported with security updates. We recommend users always keep their browser extensions automatically updated through their respective Add-on stores.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

Security is a top priority for LinguaFlow, especially because we handle user-provided API keys and DOM injection on sensitive websites. 

If you discover a security vulnerability within LinguaFlow, please **do not open a public issue**. 

Instead, please responsibly disclose it by emailing us directly at:
**[kloverdevs@gmail.com](mailto:kloverdevs@gmail.com)**

We will treat your report with the highest priority and aim to respond within 48 hours to acknowledge receipt and provide an estimated timeline for the patch.

### What to Include in Your Report
To help us resolve the issue swiftly, please provide:
* A detailed description of the vulnerability.
* Steps to reproduce the issue (including the specific translation engine and website context, if applicable).
* Any potential impact or exploit scenarios (like XSS or API key exposure).

Thank you for helping keep LinguaFlow safe!
